# Tipfy Protocol

Decentralized Donation & Staking Vault on Monad Network.

## Setup
1. Clone the repo
2. Install dependencies: `pnpm install`
3. Copy `.env.example` to `.env` and fill in your credentials.
4. Run development: `npm run dev`

## Features
- Direct P2P Donations
- Aave V3 Staking Vault
- Real-time Alerts via Ably
- AI Message Shield
